Publish Checklist

更新 Obsidian 后如何发布网站

GitHub Pages 只认识仓库里的静态文件，不能自动读取你电脑里的 Obsidian。更新笔记后，需要先同步生成网站数据，再提交并推送。

  1. 1

    在 Obsidian 里更新或新增 Markdown 笔记。

  2. 2

    在当前网站目录运行同步脚本，重新生成 content.js。

  3. 3

    用 git add、commit、push 把更新后的静态文件推到 GitHub。

  4. 4

    等待 GitHub Pages 自动重新部署，公开网站就会刷新。
