# pawn 的个人学习网站

基于 Next.js App Router、TypeScript 和 Tailwind CSS 的静态学习博客，部署在 GitHub Pages：
[https://frattyrant.github.io/study_website/](https://frattyrant.github.io/study_website/)

笔记同步

```powershell
npm run sync-content
npm run validate-content
npm run build

```

本地检查


```powershell
npm run dev
npm run lint
npm run typecheck
npm run build
```
